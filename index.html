Agent Laboratory:

Using LLM Agents as Research Assistants

Paper Code (coming soon)

Abstract

Recently, major progress has been made toward automating complex tasks that traditionally require a team of humans, including solving repository-level GitHub issues and training machine learning models to solve Kaggle challenges. These systems not only demonstrate high performance, but are also being translated into tools that improve human productivity. However, to date, there is an absence of tools that can assist human researchers similar to code assistants, such as GitHub Copilot and Cursor AI. Toward this, we introduce Agent Laboratory, an open-source large language model (LLM) agent framework for accelerating the individual's ability to perform research, specifically in the machine learning domain. Unlike previous work, where agents produce their own research topics, this work aims to function as a tool to assist human scientists to perform their own research ideas using language agents, rather than automate the entire process. Agent Laboratory takes a human-produced research idea as input and outputs a code repository and a research report. This is accomplished through the use of specialized agents driven by LLMs that collaborate to perform research based on the human's preference for the agent's involvement. In order to accommodate all users, Agent Laboratory is compute flexible, where various levels of resources can be allocated based on the individualâ€™s access to compute (e.g. CPU, GPU, memory) and model inference budget (open-source or closed-source LLM back-ends). We hope Agent Laboratory enables researchers to allocate more effort toward creative ideation and experiment design rather than low-level coding and writing, and ultimately accelerate scientific discovery.
